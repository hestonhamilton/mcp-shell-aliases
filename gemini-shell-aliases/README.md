# MCP Bash Aliases Server

This package hosts the Python implementation of the MCP Bash Aliases server built with FastMCP. See the repository root README for a full overview.

## Installing

```bash
pip install -e .[dev]
```

## Running

```bash
mcp-bash-aliases --config config.yaml --verbose
```

## Configuration

Refer to `docs/CONFIGURATION.md` for all options and defaults.

## Tests

```bash
pytest
```
